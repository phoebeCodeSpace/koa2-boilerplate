# koa2-boilerplate

## features

- [x] babel编译，支持 `import`、 `class`、`decorator` 等语法
- [x] 路由的自动加载
- [x] nodemon 与 debug 结合
- [x] 参数校验
- [x] sequelize 定义模型
- [x] error捕获与错误码设计
- [ ] jwt 权限控制
- [ ] 日志系统

## use

```bash
git clone https://github.com/phoebeCodeSpace/koa2-boilerplate.git [your project]
```
